# Ember tryout
An app that calculates the amount of ingredients you need for multiple recipies

	```
	onion soup:{onion: 1,
							veggie stock: 4
						 }
	fried rice:{onion: 1,
							carrot: 1,
							rice: 4,
							soy sauce: 1
						 }
	output => {onion: 2, veggie stock: 4, carrot: 1, rice: 4, soy sauce: 1}
	```
